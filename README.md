## Purpose

This repository presents the **screen-recorded web application demonstration** of **TriStackFuse-BD**, a calibrated multimodal speech–text fusion framework for Bengali regional dialect and Standard Colloquial Bengali classification.

The repository is prepared as supporting demonstration evidence for the manuscript:

> **Calibrated Parameter-Efficient Multimodal Speech-Text Fusion for Bengali Regional Dialect and Standard Colloquial Bengali Classification**

This repository is intended only to show how the developed real-time application works. It is **not** a source-code release, model-weight release, dataset release, or full reproducibility repository.

## Demonstrated Workflow

| Step                | Description                                                                          |
| ------------------- | ------------------------------------------------------------------------------------ |
| Input               | Bengali speech is provided through live recording or `.wav` file upload.             |
| Optional Transcript | Bengali transcript text may be added to support multimodal prediction.               |
| Inference           | The finalized TriStackFuse-BD pipeline processes the input.                          |
| Output              | The application displays the predicted dialect with confidence-ranked probabilities. |
